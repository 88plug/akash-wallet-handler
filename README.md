# akash-wallet-handler

A command line GUI for managing an Akash wallet

## Dependencies

- Docker
- JQ
- Dialog

Tested on Ubuntu 18.04+ 
```
apt-get update ; apt-get -yqq install jq dialog docker.io
```

## Getting Started

Install `jq` and `dialog`. Then `./menu.sh`.
